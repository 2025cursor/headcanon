# SEO 优化指南

## 1. Title 页面标签
- 为每个页面提供独特的标题，简洁准确地描述页面内容
- 标题长度保持在50-60字符以内（避免在搜索引擎结果中被截断）
- 将重要的关键词放在首位，但要自然，就像首先为访客编写标题一样
- 在全站所有页面的标题结尾加上网站名称和域名，如果网站名称就是域名，则只需要加域名

## 2. Description 页面描述
- 为每个页面提供独特的元描述，清晰地反映页面所传达的价值
- 字符数在150-160（包括空格）
- 包括最重要的关键词，在搜索引擎结果页面上突出显示，但要避免关键词堆砌
- 可选用引人注目的号召行动、独特主张，或关于期望的额外提示，如："学习"、"购买"等

## 3. 文本内容
- 一个页面1200个单词以上
- 核心关键词密度3%到5%
- 首先要保证自然，不要刻意堆砌关键词
- 让用户觉得这些内容就是为他写的

## 4. Heading 标题标签（H1到H6）
- 确保标题与它们描述的文本段相关
- 让标题反映其所覆盖文本的情感，避免使用简单的顺序标题
- 不要过度使用标签和其中的关键词，保持对用户的可读性
- Title 和 H1 可以一样，也可以不一样。如果写不出不一样的H1，让H1和Title一样也可以

## 5. 图片 Alt
- 正确写法：`<img src="图片地址" alt="图片描述">`
- 在正确的上下文里使用正确的图片，在图片alt里正确写图片描述
- 尽力优化最突出的图片（产品图片、信息图表或培训图片），这些图片可能会在谷歌图片搜索中被查找到
- 确保替代文本清晰且具有描述性，合理使用关键词，并确保它们自然地融入页面内容

## 6. 链接的 Nofollow 属性
- 任何可能被视为"不受信任内容"的资源链接
- 任何付费或赞助链接
- 评论或其他类型的用户生成内容中的链接可能会被滥用
- 具体用法：`<a href="链接网址" rel="noopener noreferrer nofollow">锚文本</a>`

## 7. robots 元标签
- 不是robots.txt文件，而是具体网页head里的robots meta标签
- 用法：
```html
<!DOCTYPE html>
<html><head>
<meta name="robots" content="noindex">
(...)
</head><body>
(...)
</body></html>
